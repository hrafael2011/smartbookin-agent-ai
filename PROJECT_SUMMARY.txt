                          CAPACIDADES IMPLEMENTADAS
================================================================================

Para el Cliente (WhatsApp):
✅ Agendar citas con lenguaje natural ("quiero un corte mañana a las 3")
✅ Consultar citas próximas
✅ Cancelar citas (activa waitlist automáticamente)
✅ Reagendar citas
✅ Recibir recordatorios 24h antes
✅ Confirmar asistencia 2h antes con botones
✅ Primera interacción personalizada (captura nombre)

Para el Dueño (Panel Web):
✅ Dashboard con métricas en tiempo real
✅ Login con JWT (auto-refresh tokens)
🔄 Vista de calendario (en progreso)
🔄 Gestión de citas (en progreso)
🔄 Gestión de clientes (en progreso)
🔄 Gestión de servicios (en progreso)
🔄 Configuración de horarios (en progreso)
✅ Recibir agenda diaria por WhatsApp (7 AM)

Automatizaciones:
✅ Recordatorio 24h antes (cada 15 min)
✅ Recordatorio 2h antes con confirmación (cada 5 min)
✅ Lista de espera FIFO con expiración (5 min)
✅ Notificación de cancelación al siguiente en fila
✅ Agenda diaria al dueño

================================================================================
                          TECNOLOGÍAS UTILIZADAS
================================================================================

Backend:
  • Django 5.x + Django REST Framework
  • PostgreSQL (base de datos)
  • Redis (contexto conversacional)
  • Celery + Beat (tareas programadas)

Agent Service:
  • FastAPI (webhook WhatsApp)
  • OpenAI GPT-4o-mini (NLU)
  • Redis (state persistence)
  • WhatsApp Cloud API (Meta)

Frontend:
  • React 19 + TypeScript
  • Vite (build tool)
  • TailwindCSS v4 (estilos)
  • Zustand (state management)
  • React Router v6 (routing)
  • Axios (HTTP client)
  • React Query (data fetching)

DevOps:
  • Docker + Docker Compose
  • Nginx (reverse proxy)

================================================================================
                              FLUJOS DEMOSTRADOS
================================================================================

1. AGENDAR CITA:
   Cliente: "Quiero un corte para mañana en la tarde"
   Bot: [NLU procesa] → [Normaliza fecha/hora] → [Consulta disponibilidad]
        "Encontré estos horarios:
         1️⃣ 02:00 PM ⭐
         2️⃣ 03:30 PM
         3️⃣ 05:00 PM"
   Cliente: "El primero"
   Bot: "✅ ¡Listo! Tu cita está confirmada para viernes 5 dic a las 2 PM"

2. CANCELAR CITA (CON WAITLIST):
   Cliente: "Necesito cancelar mi cita del 15"
   Bot: [Lista citas] → "¿Confirmas cancelación?"
   Cliente: "Sí"
   Bot: ✅ Cancela → 🔄 Activa waitlist → Ofrece a siguiente en fila

3. REAGENDAR CITA:
   Cliente: "Quiero mover mi cita para el sábado"
   Bot: [Selecciona cita] → "¿Para cuándo?"
   Cliente: "El sábado a las 11"
   Bot: [Consulta disponibilidad] → Ofrece slots → Actualiza cita

4. RECORDATORIOS AUTOMÁTICOS:
   24h antes: "🔔 Te recuerdo que mañana tienes tu cita..."
   2h antes: "⏰ Tu cita es en 2 horas. Confirma tu asistencia: ✅ Confirmar"

================================================================================
                              COMANDOS ÚTILES
================================================================================

Backend (Django):
  cd backend/web-backend
  python manage.py runserver          # Iniciar servidor Django
  python manage.py migrate            # Aplicar migraciones
  python manage.py createsuperuser    # Crear admin

Agent Service:
  cd backend/agent-service
  uvicorn main:app --reload           # Iniciar FastAPI

Celery:
  celery -A core worker -l info       # Iniciar worker
  celery -A core beat -l info         # Iniciar beat

Frontend:
  cd frontend
  npm install                         # Instalar dependencias
  npm run dev                         # Iniciar desarrollo (port 3000)
  npm run build                       # Build producción

Demo:
  python demo_conversation_flow.py    # Ver demostración completa
